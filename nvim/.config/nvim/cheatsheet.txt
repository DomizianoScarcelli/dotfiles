## telescope
Find Files | <leader>ff
Find Git Files | <leader>gf
Live Grep | <leader>fg
Find Recent Files | <leader>fo
Git Commits | <leader>gc
Git Status | <leader>gs

## toggleterm
Toggle Terminal | <C-\>

## tmux
Cycle screen | <prefix>n

## nvim
Search and replace | %s/search/replace

## harpoon
Open buffer | <C-e>
Goto file 1 | <C-h>
Goto file 2 | <C-j>
Goto file 3 | <C-k>
Goto file 4 | <C-l>
 
## misc
Go to next buffer | L
Go to previous buffer | H
Delete a buffer | <leader>bd
